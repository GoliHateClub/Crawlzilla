### Super Admin
- Super Admin
  - `/add_admin`
    > Add new admin to admin list
  - `/remove_admin`
    > Remove admin from admin list

### Filters
- All Users
  - `/filters`
    > Show all filters
  - `/remove_filter`
    > Remove a filter
  - `/update_filter`
    > Update a filter
  - `/add_filter`
    > Create a new filter

### Ads
- Super admin
  - `/add_ad`
    > Add new add to database 
  - `/remove_ad`
    > Remove a ad from database
  - `/update_ad`
    > Update ad information in database
- All users
  - `/see_all_ads`
    > Show adds with pagination
